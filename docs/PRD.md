## 1. 產品概述

### 1.1 產品願景

**Green購** 是一個以 ESG 永續精神為核心的企業福利平台，整合綠色採購、企業旅遊、志工活動、課程講座等服務，為企業福委會提供一站式解決方案，同時協助企業產出 ESG 相關數據報告。

### 1.2 產品定位

```
┌─────────────────────────────────────────────────┐
│  Green購 = ESG 導向的企業福利整合平台           │
├─────────────────────────────────────────────────┤
│  核心價值：                                      │
│  • 一站式服務：採購 + 旅遊 + 活動 + 報表        │
│  • ESG 數據追蹤：自動產出永續報告               │
│  • MIT 在地支持：推廣台灣優質品牌               │
└─────────────────────────────────────────────────┘
```

### 1.3 核心差異化

| 競品做法 | Green購 做法 |
|----------|--------------|
| 單純電商平台 | 電商 + 體驗服務 + ESG 報表 |
| 被動等客戶採購 | 主動提供 ESG 數據價值 |
| 價格競爭 | 價值競爭（ESG + MIT） |

## 2. 目標用戶

### 2.1 主要用戶：企業福委會

| 用戶角色 | 需求 | 痛點 |
|----------|------|------|
| **福委會主委/委員** | 規劃年度福利、控制預算 | 供應商分散、比價耗時 |
| **人資/總務** | 執行採購、處理行政 | 報表整理麻煩、ESG 數據難追蹤 |
| **企業主管** | ESG 報告、員工滿意度 | 缺乏數據支持 |

**目標企業規模**：50-2000 人，有福委會組織

### 2.2 次要用戶：供應商/廠商

| 用戶角色 | 需求 | 痛點 |
|----------|------|------|
| **MIT 品牌商** | 接觸企業客戶 | B2B 通路有限 |
| **在地小農** | 穩定訂單 | 行銷資源不足 |
| **活動講師** | 企業內訓機會 | 難以接觸 HR |

### 2.3 用戶旅程

```
企業福委會發現 Green購
        ↓
註冊成為企業會員（免費）
        ↓
瀏覽商品/服務 → 詢價/下單
        ↓
平台自動記錄 ESG 數據
        ↓
月度/年度 ESG 報表產出
        ↓
企業依賴報表 → 持續使用（鎖客）
```

## 7. 資料模型（核心）

### 7.1 主要實體

```
Company（企業）
├── id, name, tax_id, address
├── plan_type (free/basic/pro)
├── contact_person, email, phone
└── created_at, updated_at

User（用戶）
├── id, company_id, email, password
├── role (admin/buyer)
├── name, phone
└── created_at, updated_at

Vendor（廠商）
├── id, name, tax_id
├── status (pending/approved/rejected)
├── commission_rate
└── contact_info, bank_info

Product（商品）
├── id, vendor_id, name, description
├── category, tags
├── price, stock, status
├── images, specs
└── esg_tags (MIT/organic/eco...)

Order（訂單）
├── id, company_id, user_id
├── status, total_amount
├── shipping_info, payment_info
└── created_at, updated_at

OrderItem（訂單明細）
├── id, order_id, product_id
├── quantity, unit_price
└── vendor_id

Activity（活動/旅遊）
├── id, title, type (travel/volunteer/course)
├── description, schedule
├── price_range, capacity
└── status, images

Inquiry（詢價單）
├── id, company_id, activity_id
├── requirements, budget
├── status, response
└── created_at

ESGRecord（ESG 紀錄）
├── id, company_id, record_type
├── value, unit
├── source (order_id/activity_id)
└── recorded_at
```

## 11. 附錄

### 11.1 競品參考

| 平台 | 特點 | 我們的差異 |
|------|------|-----------|
| 福委好幫手 | 商品齊全、價格低 | 我們有旅遊/活動 + ESG |

### 11.2 ESG 報表欄位參考

**環境面 (E)**
- MIT 在地採購金額
- 環保標章商品採購比例
- 低碳旅遊減碳量估算
- 減塑包裝節省量

**社會面 (S)**
- 志工服務總時數
- 員工活動參與率
- 支持社企/小農數量
- 公益捐贈金額

**治理面 (G)**
- 採購流程透明度
- 供應鏈 ESG 評估
